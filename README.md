> 不是"记笔记"，而是"构建一个能反复利用的知识系统"

## 目录结构

```
knowledge-base/
├── AI/              # AI 工具与技术
├── Backend/         # 后端开发
│   └── Spring/      # Spring 框架
├── ComputerScience/ # 计算机基础
├── Thinking/        # 思维方法论
└── README.md
```

## 内容原则

- ❌ 不写"今天学了什么"
- ✅ 写"这个东西是什么 + 怎么用"
